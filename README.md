 ## Requirements to run the Project 
 - Basic JavaScipt Knowledge
 - Confortable with REACTJS framework
 - Code Editor like Visual Studio code
 - GitHub account

## Getting Started
 - git init
 - Clone React Front-end from (https://github.com/yusufnviiri/dsaccofront.git)
 - cd dsaccofront
 - Run npm i
 - Run npm start 
 - Enjoy
